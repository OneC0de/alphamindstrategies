## Alpha Mind Strategies Marketing Site

This repository contains the Alpha Mind Strategies marketing experience built with Next.js 16, TypeScript, and Tailwind CSS.

### Development

```bash
npm install
npm run dev
```

The site lives in `src/app/page.tsx`, with shared styles defined in `src/app/globals.css`. Update sections there to evolve messaging.

### Build

```bash
npm run build
npm start
```

### Deployment

Deploy via Vercel (recommended) or any platform that supports the Next.js App Router. Push the `main` branch to kick off the Vercel build.
