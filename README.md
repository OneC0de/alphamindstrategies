## Alpha Mind Strategies Marketing Site

This repository contains the Alpha Mind Strategies experience, built with Next.js 16, TypeScript, and Tailwind CSS.

### Development

```bash
npm install
npm run dev
```

Both dev and production preview run on [http://localhost:3001](http://localhost:3001). The site lives in `src/app/page.tsx`, with shared styles defined in `src/app/globals.css`.

### Build

```bash
npm run build
npm start
```

### Deployment

Deploy via Vercel. Push the `main` branch to kick off the Vercel build.
